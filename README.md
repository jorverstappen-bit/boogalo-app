# Boogalo — TBC Raid Manager

A World of Warcraft: The Burning Crusade raid management app.

## Deploy to Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/jorverstappen-bit/boogalo-app)

Or drag the folder into [app.netlify.com/drop](https://app.netlify.com/drop).

## Local development

Just open `index.html` in your browser — no build step needed.
